## Installation

1. Install zsh `brew install zsh`
3. Clone this repo into your `$HOME`-Folder with `git clone --recurse-submodules --remote-submodules`
4. Other:
    - `brew install pandoc`
    - `brew install cask`
    - `brew install tmux`
    - `brew install vim`
    - In vim `:PlugInstall`
    - `brew install pandoc-citeproc`
    - `brew install python`
    - `pip install pandocfilters`
    - `brew cask install mactex`
    - `brew install xpdf`
